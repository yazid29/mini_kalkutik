LUAS SEGITIGA

Rumus Luas Segitiga Yaitu :

L = 1/2 x a x t

Dimana :
L = Luas
a = Panjang Alas
t = Tinggi

Hitung Luas Segitiga




KELILING SEGITIGA

Rumus Keliling Segitiga Yaitu :

K = S1 + S2 + S3

Dimana :
K = Keliling
S1 = Sisi A
S2 = Sisi B
S3 = Sisi C

Hitung Keliling Segitiga
